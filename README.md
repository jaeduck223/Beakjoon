# 백준 문제풀이
- 인터넷을 찾아 보거나, 가끔씩은 지피티에게 물어보곤 하는데.... 그러는 나의 모습을 보면 현타가 가끔씩 온다.......
- 코드 문법 같은 거는 인터넷에서 찾아볼 예정! (파이썬 문법을 기반으로 진행하는데는 한계가 있음.) gpt 안 쓸 거임!
- 문제 풀이는 주피터 노트북을 이용해서 진행할 것이며, 이유는 풀이 과정 기록 관리와 문제를 풀면서 느낀 점 및 오답노트 작성에 용이하기 때문임.
- 시간에 상관하지 않고 매일 10분씩이라도 꾸준히 풀려고 함.
- 문제의 난이도와 상관 없이 브론즈 난이도이더라도 다 배울 점이 있으므로 하나하나 뜯어볼 예정
- 주로 브론즈와 실버를 통해서 자신감과 재미를 챙기고, 골드 문제를 통해서 나의 한계를 느낄 것!
